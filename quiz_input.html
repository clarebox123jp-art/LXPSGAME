📚 LXPSGAME 題庫編輯器 老師上傳專用 v1.0

👋 歡迎使用！這是給老師、家長、設計者填寫題庫的工具

填好的題目交給管理員後,管理員會把題庫整合進遊戲,讓全校玩家都能挑戰你出的題目。流程很簡單:
  1. 填寫「基本資訊」(出題者姓名、科目、想加強複習的日期區間⋯)
  2. 上傳檔案匯入(支援 .csv / .xlsx / .json / .txt),或在右下方一題一題手動新增
  3. 檢查左側列表沒有紅色錯誤,按右上「📋 產生題庫單」
  4. 按「📎 複製到剪貼簿」,把整段文字貼給管理員(或老師代為轉交)

📋 基本資訊

會印在題庫單上,讓管理員知道這份題庫是誰提供的
這份題庫的整體主題,玩家答題時會看到
玩家會在符合難度的關卡裡看到這些題目
📅 加強複習提醒區間(重要!玩家在這段時間做題會獲得獎勵)
→
用途:例如下週要小考,可以設「下週一到下週五」,讓玩家收到「加強複習」提醒, 在這段期間答對你的題目會額外加金幣/經驗,鼓勵他們積極複習。 建議區間 3~14 天,太長提醒效果會弱化。
